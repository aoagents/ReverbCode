The runtime is where your agent's terminal actually lives. In the current Go rewrite, AO wires the Zellij runtime.

| Runtime  | Best for                                     | Binary needed   |
| -------- | -------------------------------------------- | --------------- |
| `zellij` | Current Go rewrite on macOS, Linux, Windows | `zellij` 0.44.3+ |

## Choosing

- Install `zellij` 0.44.3+ on `PATH` before spawning sessions.
- `runtime: process` is not currently wired in the Go rewrite.

The dashboard terminal connects to the daemon's Zellij-backed terminal mux.
